## Module Summaries

### Task 1 – Digital Literacy Awareness Infographic (Module 1)
Created a one-page infographic using **Canva** covering three key topics: what digital literacy means, useful digital tools for students, and safe internet practices. Designed to be shared with batchmates as a quick visual reference.

### Task 2 – Student Digital Portfolio (Module 2)
Set up professional accounts on **GitHub**, **LinkedIn**, and **Kaggle**. Created a GitHub profile README and filled in the LinkedIn Education section. These platforms will form the foundation of my professional online presence throughout my B.Tech journey.

### Task 3 – Coding & Collaboration Platforms (Module 3)
Completed a beginner challenge on **HackerRank** and built a 5-question **Digital Literacy Awareness Quiz** using Google Forms with an auto-linked response spreadsheet.

**Google Form Quiz Link:** https://docs.google.com/forms/d/e/1FAIpQLScAHu7iJO3DdxNVi9dIdxsULLFUFdY_BTj9S3ecBdQvhlQ3GQ/viewform?usp=header

### Task 4 – Email Etiquette (Module 4)
Drafted two professional emails — one requesting an assignment extension from a professor, and one expressing interest in a summer internship. Also created a Social Media Do's and Don'ts checklist for responsible online conduct.

### Task 5 – Cybercrime Awareness (Module 5)
Researched **cyberbullying** as a cybercrime type. Produced a detailed case study describing how cyberbullying typically unfolds in a college environment, and a practical prevention checklist with UPI/financial safety tips and reporting resources.
